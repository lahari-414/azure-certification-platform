How to Get Your FREE AI API Key

Using Google Gemini AI — Free forever for personal projects

1

Open Google AI Studio in your browser

Open Chrome or any browser. In the address bar at the top, type this website and press Enter:

https://aistudio.google.com/apikey
⚠️ You must be signed in to your Google account (Gmail). If not, it will ask you to sign in — just use your normal Gmail email and password.
↓
2

Click "Create API Key" button

After the page loads, you will see a page titled "API keys".

Look for a blue button that says "Create API key" and click it.

📄 The page looks like this:

[ API keys ]

[ + Create API key ] ← CLICK THIS BUTTON

(It may also say "Get API key" — click whichever you see)
↓
3

Select a project (or create one)

A small popup will appear asking you to choose a Google Cloud project.

If you have never used Google Cloud, click "Create API key in new project".

If you already have a project, just select any project from the list and click "Create API key in existing project".

⚠️ Don't worry — this is completely free. Google Cloud project is just a container to organize your key. No payment required.
↓
4

Copy your API Key

Your API key will appear on screen. It looks like this:

AIzaSyABCDEFGHIJKLMNOPQRSTUVWXYZ123456789

Click the Copy icon next to the key. Then open Notepad on your computer and paste it there so you don't lose it.

✅ Your key always starts with "AIza" — if you see that, it is correct!
↓
5

Open app.js and paste your key

Go to your aws-cert-platform folder on Desktop. Right-click app.js → Open With → Notepad.

Press Ctrl + F to open Find. Search for:

YOUR_GEMINI_API_KEY_HERE

Select that text and replace it with your actual key. It should look like:

const GEMINI_API_KEY = "AIzaSyABCDEFGHIJKLMNOPQRSTUVWXYZ123456789";

Press Ctrl + S to save. Close Notepad.

⚠️ Keep the quote marks " " on both sides of your key. Only replace the text inside the quotes.
↓
6

Run your project!

Go to your aws-cert-platform folder. Right-click index.html → Open With → Chrome.

Your AWS Certification Platform will open in the browser with FREE AI-generated questions!

✅ When you click "Start Certification Test", Gemini AI will generate 5 fresh questions every single time — completely free!

🎉 You're All Set!

Your project will generate unique AWS questions every test attempt.
